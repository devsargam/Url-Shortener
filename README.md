# url-shortner

Url shortner backend application like [Bitly](https://bit.ly)

### Requirements

- Nodejs
- Mongodb
- yarn (prefered package manager)
- git

### Steps to run locally

```bash
git clone https://github.com/devsargam/url-shortner.git
cd url-shortner
cp .env.example .env
yarn
yarn dev
```

NOTE: Setup environment variable in .env
